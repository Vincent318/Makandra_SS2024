Dieses Projekt entstand im Sommersemester 2024 in einem 5-köpfigen Team. Ziel war es, ein System zur
Verwaltung von Mitarbeitern, Rollen, Prozessen und Aufgaben für die Firma Makandra in Augsburg zu entwickeln.
Die Software unterstützt Administratoren und Benutzer bei der Organisation von Prozessen und Aufgaben und erleichtert
die tägliche Verwaltung.

Das System ermöglicht das Anlegen und Verwalten von Benutzern, Abteilungen, Rollen, Aufgaben und Prozessen.
Administratoren können Aufgaben erstellen, Deadlines setzen, Zuständigkeiten vergeben und Prozesse starten oder
archivieren. Benutzer können ihre Aufgabenlisten einsehen, Aufgaben erledigen und nach Prozessen filtern. Die
Software prüft Berechtigungen und stellt sicher, dass nur autorisierte Benutzer Zugriff auf bestimmte Funktionen haben.

Technologien: C# / .NET 6, ASP.NET Core MVC, Entity Framework, SQL-Datenbanken.

########################################################################################################################

This project was developed in the summer semester of 2024 by a team of five members. The goal was to create a system
for managing employees, roles, processes, and tasks for the company Makandra in Augsburg. The software supports
administrators and users in organizing processes and tasks and simplifies daily management.

The system allows creating and managing users, departments, roles, tasks, and processes. Administrators can create
tasks, set deadlines, assign responsibilities, and start or archive processes. Users can view their task lists,
complete tasks, and filter them by processes. The software checks permissions and ensures that only authorized
users have access to specific functions.

Technologies: C# / .NET 6, ASP.NET Core MVC, Entity Framework, SQL databases.

Quick Start:

cd Projekt/Replay

dotnet restore

dotnet run

The project demonstrates typical workflows in enterprise management, such as onboarding, project management,
and administrative processes. Data validation and permission checks are integrated to ensure consistency and
security.
